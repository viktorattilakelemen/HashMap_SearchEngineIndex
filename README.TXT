PROJECT TITLE: SearchEngineIndex
PURPOSE OF PROJECT: Demonstrate knowledge of HashMaps
VERSION or DATE: 5/9/2018
HOW TO START THIS PROJECT: Compile all java files and specify class path
(ex: javac -cp "C:\V\Block 8_CS2\SearchEngineIndex_Assignment\jsoup.jar"
 *.java), then run SearchEngineDriver similarly, specifying the class
path for jsoup.jar and the class files (ex: java -cp "C:\V\Block 8_CS2\
SearchEngineIndex_Assignment\jsoup.jar;." SearchEngineDriver)
AUTHORS: Brianna Fuentes, Viktor Kelemen
USER INSTRUCTIONS: input a search word when prompted
---------------------------------FILES----------------------------------
QueryResult - defines QueryResult objects which store all information
connected to one paricular search result
SearchEngineDriver - holds the main class which handles user input, error
message and method calls
SearchEngineIndex - holds the bulk of methods which read in html files
from the specified directory, creates and fills hashmap, returns it, etc
--------------------------------DISCLAIMER-------------------------------
The program might take more than 10 seconds to build the initial hashmap,
before displaying the input prompt!
